### 3. Cosa può fare Claude Code una volta connesso

Con questo MCP Claude Code può:

- **Eseguire comandi** sul pod: `tail -f /workspace/comfyui.log`, `ps aux`, `df -h`
- **Leggere log** in tempo reale: ComfyUI log, api_server log, setup log
- **Modificare file** direttamente su `/workspace/upscaler/*.py`
- **Testare le API** con curl direttamente dal pod (localhost, nessun proxy)
- **Riavviare servizi**: `pkill -f api_server.py && python3 api_server.py &`
- **Debug automatico**: legge l'errore → modifica il file → riavvia → rilegge il log
